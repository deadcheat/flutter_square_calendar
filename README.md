# square_calendar

A widget plugin for showing calendar.

%%% NOTIFICATION %%%
In this version, you can only show calendar yet.

## Getting Started

```
import 'package:square_calendar/square_calendar.dart';
```

```
new SquareCalendar(year: 2018, month: 2, day: 1,),
```
